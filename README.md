# 👁️ Kind Eye — Children’s Vision Protection App

**Kind Eye** is a simple and helpful app designed to protect children’s eyesight while using digital devices.

---

## 🚀 Features

* 📏 **Distance Monitoring**
  Tracks the distance between the eyes and the screen using the front camera.

* ⏰ **Break Reminders**
  Encourages regular breaks to reduce eye strain.

* 🧘 **Eye Exercises (Optional)**
  Guides users through short eye exercises before each break.

# 👁️ Kind Eye — Приложение для защиты зрения детей

**Kind Eye** — это простое и полезное приложение, созданное для защиты зрения детей при использовании цифровых устройств.

---

## 🚀 Возможности

* 📏 **Отслеживание расстояния**
  Контролирует расстояние от глаз до экрана с помощью фронтальной камеры.

* ⏰ **Напоминания о перерывах**
  Помогает регулярно делать перерывы, снижая нагрузку на глаза.

* 🧘 **Зарядка для глаз (по желанию)**
  Предлагает короткие упражнения для глаз перед каждым перерывом.
